# newdtbuild

Production Daily Thunder automation repository.

## What this repo does
- Runs production Daily Thunder automation.
- Maintains the central `season_state.json` and `content_state.json` registry.
- Hosts workflows for:
  - Update DT State
  - Pregame
  - Scoreboard
  - Day After
  - Podcast

## Archive / yearbook
Archive and yearbook workflows and source tooling live in `dailythunder/dt-archive-private`.
